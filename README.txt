To run this program, run the file 'CS476 P2.cpp', to do so, the following commands can be used
g++ 'CS476 P2.cpp'
./a.out

When the program is executed, the user will be prompted to enter a filename, and if a valid file
is entered the user will be prompted to select the algorithm they would like to run on that tables
data values. The following user input corresponds to the algorithm that will be run:
u - UPGMA algorithm
n - Neighbor Joining algorithm
